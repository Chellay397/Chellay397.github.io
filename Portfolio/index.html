DUH TRENDS PALAS WEB - Project Details

This project introduces a web-based Point-of-Sale (POS) and Inventory Management System for Duh Trends Palas...

Features

  • POS System: Processes sales transactions and updates inventory in real time.
  • Order Website Integration: Orders from the POS are sent to the online order management system.
  • Product Management: Generates unique barcodes for new products.
  • Contract and Storage Type Integration: Links contracts to specific storage types for better organization.
  • Employee Tracking: Records which employee operates the POS system.

Project Gallery

The homepage showcases the POS system, where users can scan product barcodes stored in the database

The Orders page displays processed transactions with a view details option to check customer purchases.

The Products section allows users to generate a unique barcode for new items.

On the contract page, the user can input the date, month, and year for the contract's start and end dates. Additionally, the Brand Partner ID is entered on this page.

The Contracts section links with storage types, ensuring proper inventory categorization.

The Employees module enables store management by adding and maintaining employee records.
